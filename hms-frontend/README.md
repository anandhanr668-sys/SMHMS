# HMS Frontend

> Developer note: In development the app will auto-set `tenantId` in localStorage to `local` so LCNC pages and API calls work without logging in. Toggle this behavior by clearing localStorage or setting a real tenant id during testing.


Frontend application for the Hospital Management System (HMS).

## Tech Stack
- React 18
- Vite
- TypeScript
- React Router
- Axios
